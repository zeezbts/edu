INHERITANCE

  • In inheritance allows creating a logical relationship between two or more class
  • Depending on the number of class involved and the way the class are inherited

    • TYPES OF INHERITANCE

  • TYPES
    1. single inheritance
    2. multiple inheritance
    3. hybrid inheritance
    4. multilevel inheritance

    1.single inheritance

  • when a drived class inherits from a single base class , it is known as single inheritance
  • The drived class inherits all the members of it's base class and can directly access the public and protected members of the base class

    • syntax

    class derived_class:access specifier base_class

    {

    //body;

    };

    9. }

    15. {

    18. };

    Hybrid inheritance

  • When two or more types of inheritance are combined in one program.

    • syntax

  • class A
    {
    members
    };
    class B : public A
    {
    members
    };
    class C : public A
    {
    members
    };
    class D : public B, public C
    {
    members
    };

    • example

  • #include <iostream.h>
    #include <conio.h
    class Parent
    {
    public:
    void parentMsg()
    {
    cout <<"Function from Parent";
    }
    };
    class Base1 : public Parent
    {
    public:
    void show1()
    {
    cout << "Inside Base1";
    }
    };
    class Base2
    {
    public:
    void show2()
    {
    cout << "Inside Base2";
    }
    };
    class Derived : public Base1, public Base2
    {
    public:
    void display()
    {
    cout << "Inside Derived Class";
    }
    };
    void main()
    {
    clrscr();
    Derived d;
    d.parentMsg();
    d.show1();
    d.show2();
    d.display();
    getch();
    }

    • Multilevel inheritance

  • Multilevel inheritance in C++ means a class is derived from another derived class, forming a chain of inheritance.

    • syntax

  • class BaseClass
    {
    // Base class members
    };
    class derived_class1: access_specifier BaseClass
    {
    // Derived members
    };
    class derived_class2: access_specifier baseclass2
    {
    // Derived from IntermediateClass
    };

    • example

  • #include <iostream.h>
    #include <conio.h>
    class baseclass
    {
    public:
    void get()
    {
    cout << "Animal is eating..." << endl;
    }
    };
    class derived1: public baseclass
    {
    public:
    void put()
    {
    cout << "Dog is barking..." << endl;
    }
    };
    class derived2: public derived2
    {
    public:
    void out()
    {
    cout << "Puppy is weeping..." << endl;
    }
    };
    void main()
    {
    clrscr();
    derived2 obj;
    obj.get();
    obj.put();
    obj.out();
    getch();
    }
