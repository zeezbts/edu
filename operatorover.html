OPERATOR OVERLOADING

  • Operator overloading is a compile time polymorphism in which the operator overload to provoide the special meaning to the user defined data type
  • operator overloading is used to overload most of operators available in c++
  • it is used to perform the ooperation on the user defined data type

    • Operator that cannot be overload:
